##Tic Tac Toe Game

#Details
The Tic-tac-toe game is develop with java swing, using NetBeans IDE 7.4.
The game is for two players, Computer (C) and user player (P), who take turns marking 
the spaces in a 3×3, 4X4 and grid. The player who succeeds in placing three of their marks
in a horizontal, vertical, or diagonal row wins the game. The computer player always play the best move and never lose.

#Development/Configuration
 The following where use in the development
 1. NetBeans IDE 7.4
 2. JDK and JRE
 
 #Deployment
 1. Install NetBeans IDE 7.4 and JDK and JRE. 
 2. Clone or download the project and save it in your netbeans project folder or any where in your computer.
	Exaample: C:\Users\SULEIGOLDEN\Documents\NetBeansProjects
 3. Open your NetBeans IDE.
 4. Click on file and click on open project.
 5. Navigate to your netbeans project folder.
 6. Click on the project that you want to open (Tic Tac Toe).
 7. Click open project.
 8. Click on the green play run button at the top of the netbeans menu bar to rum the game.
 9. Enjoy.
